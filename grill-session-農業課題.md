# グリルセッション：農業の作業課題解決ツール

日時: 2026-05-11 / 2026-05-13

---

## 決まったこと

- **対象ユーザー**: 個人農家〜農業法人（両方）
- **農業種別**: 水田・穀作を起点に設計
- **最大の痛み**: ① 農場ごとの作業スケジュール管理、② 日報・作業日誌の記入
- **現状**: 紙・Excel・LINE がバラバラに混在
- **チーム規模**: 2〜5人（家族経営・小規模農業法人）
- **端末**: スマホ中心
- **成功の定義**: 誰が何をやっているか一目でわかる（見える化）
- **差別化ポイント**: シンプルさ（既存ツールより迷わず使える）
- **競合との関係**: アグリノート・クボタKSASHA は API 非公開 → スタンドアロンで独立
